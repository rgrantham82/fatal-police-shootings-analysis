# GitHub Upload Instructions

## Option A: Upload with GitHub web UI

1. Create a new GitHub repository.
2. Name it something like `fatal-police-shootings-analysis`.
3. Keep it public if you want employers to see it.
4. Upload the contents of this folder.
5. Pin the repository to your GitHub profile.

## Option B: Upload from command line

From inside this folder:

```bash
git init
git add .
git commit -m "Initial fatal police shootings analysis portfolio repo"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/fatal-police-shootings-analysis.git
git push -u origin main
```

## Suggested repository description

Per-capita, statistical, predictive, and geographic analysis of fatal police shootings in the U.S., with focused New Mexico and Albuquerque outlier analysis.
